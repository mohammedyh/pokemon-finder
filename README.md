- Search functionality
- Pagination
- Debouncing
- Responsive
- Loading states

## Run Locally

Clone the project:

```bash
git clone https://github.com/mohammedyh/pokemon-finder.git
```

Go to the project directory

```bash
cd pokemon-finder
```

Install dependencies

```bash
pnpm install
```

Start the server

```bash
pnpm dev
```
