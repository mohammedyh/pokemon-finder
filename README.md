## Features

- Search functionality
- Pagination
- Debouncing
- Responsive
- Loading animations

## Run Locally

Clone the project:

```bash
git clone https://github.com/mohammedyh/pokemon-app.git
```

Go to the project directory

```bash
cd pokemon-app
```

Install dependencies

```bash
pnpm install
```

Start the server

```bash
pnpm dev
```
